Instrucciones para Implementar el Proyecto de Enviar email en Java con Conexión a Base de Datos en MySQL del Capìtulo 14 del Curso de Java y MySQL

1. Descargar Librería Java Mail desde http://www.oracle.com/technetwork/java/javamail/index-138643.html

2. Importar el proyecto que es la carpeta "correos" a Netbeans.

3. Agregar a las bibliotecas del proyecto las siguientes librerías:

 • mail.jar
 • mysql-connector-java-version.jar

4. Cargar la Base de Datos a MySQL. (correos.sql).

5. Editar las constantes de conexión en la Clase Conexion.java por nuestras credenciales de MySQL

public static final String URL = "jdbc:mysql://localhost:3306/correos"; //Dirección, puerto y nombre de la Base de Datos
public static final String USERNAME = "root"; //Usuario de Acceso a MySQL
public static final String PASSWORD = ""; //Password del usuario

6. Ejecutar la Aplicación

7. Agradecer.
